Как это работает

Чтобы начать трансляцию своего канала, тужен только экран и популярный микрокомпьютер Raspberry Pi 3 model B.

Вы можете заказать Raspberry Pi 3 model B у официального дилера, в интернет-магазине или купить в местном магазине электроники со специализацией "сделай сам".

Блок питания, флэш-карта и корпус могут продаваться отдельно.

Для подключения Raspberry Pi 3 model B к Cloud TV следуйте инструкциям в личном кабинете Cloud TV.
